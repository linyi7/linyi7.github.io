## Markdown基础语法总结

1.标题

类Setext

This is an H1

This is an H2

  Code:

  This is an H1   
  ====
  This is an H2
  ----
备注:任何数量的=和-都可以有效果<br/>
=表示最高阶标题,-表示第二阶标题

2.atx形式

#This is an H1

##This is an H2

###This is an H3

####This is an H4

#####This is an H5

######This is an H6
---
  Code:

  # This is an H1

  ## This is an H2

  ### This is an H3

  #### This is an H4

  ##### This is an H5

  ###### This is an H6
---
也可以表示成这样 # This is an H1 #
备注: 行首n个#表示n阶标题，n最大为6
